# AI Customer Support Dashboard

This project is an AI-powered customer support analytics dashboard.

Features:
- Live ticket data from Google Sheets
- Automated support insights
- Chatbot to query ticket data
- Interactive dashboard visualization
- Embedded Looker Studio analytics

Tech Stack:
- Gradio
- Data Automation
- Google Sheets Integration
- Support Analytics
- Dashboard Reporting

Use Case:
Helps support teams quickly understand ticket trends, open issues, urgency levels, and common problems.
